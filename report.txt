🧠 Introduction
This report documents an OSINT (Open Source Intelligence) investigation conducted as part of a capstone challenge from Security Blue Team.
The objective was to track and build a complete profile of a fictional Person of Interest (POI) using only publicly available information, starting with a single clue — a Twitter handle.
🎯 Initial Clue
Twitter Handle: @sp1ritfyre
🔍 Methodology & Investigation Steps
🔹 Step 1: Twitter Recon
I began by locating the Twitter profile using Google dorking:
inurl:@sp1ritfyre site:twitter.com
The profile contained a Base64 encoded link.

🔹 Step 2: Base64 Decoding
The encoded string was decoded using a Base64 decoder.
➡️ Result: redhunt.net

🔹 Step 3: URL Analysis
Before visiting the site, I analyzed it using:
urlscan.io
VirusTotal
📌 Findings:
Hosted on AWS EC2 (UK)
No malicious activity detected

🔹 Step 4: Expanding Search
Performed Google dorking:
intext:@sp1ritfyre
➡️ Found a Blogger account linked to the same username.

🔹 Step 5: Blogger Analysis
The Blogger profile contained:
Email address
A suspicious hex-encoded location field

🔹 Step 6: Hex Decoding
The hex string was decoded using a hex-to-text tool.
➡️ Result: sammiewoodsec.blogspot.com

🔹 Step 7: Final Profiling
By correlating data from multiple sources, a complete profile was built.
